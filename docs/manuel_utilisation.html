Manuel d'Utilisation - Esportify

Version : 1.0

Date : Mai 2026

1. Présentation de l'application

Esportify est une plateforme web permettant de découvrir, créer et participer à des tournois et compétitions e-sport. Trois types d'utilisateurs cohabitent :

  • Le Visiteur : consulte les événements publics sans pouvoir s'inscrire.
  • Le Joueur : s'inscrit aux tournois, gère ses favoris et consulte ses scores.
  • L'Organisateur : crée des événements, modère les inscriptions et lance les sessions.
  • L'Administrateur : valide les événements, gère les utilisateurs et supervise la plateforme.

2. Accès et connexion

Accédez à la plateforme via l'URL de déploiement. Le menu principal permet la navigation entre l'accueil, la liste des événements, et la connexion.

3. Parcours Joueur

  1. Inscription : Cliquez sur "Inscription" dans le menu, remplissez le formulaire (pseudo, email, mot de passe).
  2. Connexion : Connectez-vous avec votre email et mot de passe.
  3. Consulter les événements : Allez dans "Événements". Utilisez les filtres pour trier par date, nombre de joueurs ou organisateur.
  4. S'inscrire : Cliquez sur "Voir les détails" puis "S'inscrire". Votre inscription sera en attente de validation.
  5. Espace personnel : Dans "Mon espace", consultez vos inscriptions, vos favoris, votre historique d'événements proposés et vos scores.
  6. Rejoindre un événement : Si l'événement a démarré et que vous êtes accepté, un bouton "Rejoindre" apparaît.

4. Parcours Organisateur

  1. Demande de rôle : Par défaut, un administrateur doit promouvoir votre compte en "organisateur". Connectez-vous en admin pour le faire (ou utilisez le compte org1/org2).
  2. Créer un événement : Dans "Mon espace", remplissez le formulaire "Créer un événement". L'événement passe en attente de validation.
  3. Gérer les inscriptions : Dans "Espace Organisateur", acceptez ou refusez les joueurs inscrits à vos événements.
  4. Démarrer un événement : 30 minutes avant l'heure prévue, un bouton "Démarrer" apparaît dans "Mes événements".
  5. Modifier un événement : Tant que la date n'est pas passée, vous pouvez modifier votre événement. Il sera alors soumis à une nouvelle validation.

5. Parcours Administrateur

  1. Connexion : Utilisez le compte admin@esportify.fr.
  2. Tableau de bord : Visualisez les statistiques globales (utilisateurs, événements, inscriptions).
  3. Modération : Validez ou rejetez les événements en attente. Suspendez un événement à tout moment.
  4. Gestion des utilisateurs : Promouvez un joueur en organisateur, rétrogradez-le, ou désactivez un compte.
  5. Ajout de scores : Attribuez des scores aux joueurs pour les événements passés.

6. Sécurité et RGPD

  • Les mots de passe sont stockés de manière sécurisée (hashage bcrypt).
  • Les données personnelles (email, pseudo) ne sont pas partagées publiquement.
  • Chaque action sensible (inscription, modification, suppression) est protégée par un token CSRF.
  • Pour demander la suppression de votre compte, contactez l'administrateur via la page "Contact".

7. Déploiement

L'application est développée en PHP et peut être déployée sur tout hébergement mutualisé ou cloud supportant PHP et MySQL. Pour le déploiement :

  1. Uploader les fichiers sur le serveur.
  2. Importer le fichier SQL dans votre base de données.
  3. Modifier les fichiers de configuration avec vos identifiants.
  4. Assurez-vous que les sessions PHP sont activées.
